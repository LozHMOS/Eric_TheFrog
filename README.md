# Underground Development Productivity Toolkit

A professional, neutral demonstration application that assists underground coal development teams to identify, analyse and implement process improvement opportunities within their own panels.

The tool is designed as a lightweight bolt-on that development crews, deputies and engineers can use daily. It embeds the core principles of structured Process Improvement Events (PIEs), time-and-motion analysis, pillar-cycle optimisation and visual performance tracking.

## Features
- Diagnostic Scanner – rapid opportunity identification
- Golden Meter Builder – time-and-motion analysis and cut-rate calculation
- Pillar Cycle Simulator – what-if scenario modelling
- Live Takt Planner – weekly shift-by-shift planning
- Performance Dashboard & History – visual metrics and trends
- Idea Bank & Action Tracker – operator-driven continuous improvement
- Future PIE Generator – suggested next improvement events
